Вопросы и ответы

Есть ли у вас в продаже плоскорезы Фокина?
В продаже у нас имеется Аэратор для грунта Ecotec c металлическим черенком и Мотыга комбинированная Ecotec.
Как правильно подобрать размер кровельного самореза? (клиент Стройоптторг Волковыск)
Длина кровельного самореза выбирается из расчета: толщина материала кровли + толщина подконструкции + длина сверла самореза (8 или 14,5 мм) + ширина шайбы EPDM (3 мм).
Какие валики больше подходят для акриловых и масляных красок?
Для акриловых красок на водной основе подойдут любые валики.
Для масляных красок лучше использовать валики из полиамида, полиакрила или натурального меха.
Следует отметить, что валик нужно выбирать также исходя из фактуры окрашиваемой поверхности.Для гладких поверхностей применяется валик с коротким ворсом, для грубых — с более длинным, для очень грубых существуют специальные фасадные валики с поролоновой подкладкой.
Каким инструментом можно воспользоваться для установки столбиков на приусадебном участке?
Для установки столбиков, столбов, посадки деревьев удобно использовать Ямкоделатель Ecotec.
Кардинальное отличие крепежа Шуруп для оконных рам от Металлический рамный дюбель?
Шуруп для оконных рам можно использовать для крепления в мягкие материалы, в частности, в газосиликат.
Причем предварительное сверление не требуется.
Необходимо прикрепить оконную раму (рама выступает из плоскости здания) из металлических труб 60х3 мм к стойке 60х4 мм через сендвич-панель толщиной 100 мм. Необходим саморез длинной 180 – 200 мм — что посоветуете/предложите?
1. Саморез для сэндвич-панелей 6,3/5,5*155 РТ5
2. Саморез для сэндвич-панелей 6,3/5,5*180 РТ5
Подскажите, пожалуйста, для каких целей используются терки с резиной и губкой? Спасибо.
Тёрка с поролоном (кубическая насечка позволяет удерживать дополнительное количество жидкости) - используется при полировке поверхностей в штукатурных работах, а также для снятия с поверхности керамической плитки лишнего слоя затирочной массы.
Тёрка с натуральной резиновой губкой - предназначена для смачивания различных поверхностей при работе с гипсовой штукатуркой.
Тёрка с чёрным каучуком предназначена для штукатурных финишных работ.
Чем можно закрепить плинтус в панельном доме?
Наиболее простой вариант – с помощью дюбель-гвоздей через отверстие в плинтусе в предварительно просверленное отверстие в бетоне.
Чем отличается пенопластовая терка от полиуретановой?
Полиуретановая терка плотнее пенопластовой и, соответственно, более износостойкая. Для разовых работ рациональнее использовать пенопластовую терку.
Чем отличается шуруп от самореза?
Разница между шурупом и саморезом заключается в том, что шурупы требуют предварительно просверленного отверстия, а саморезы подходят практически для всех материалов и не требуют предварительной подготовки рабочей поверхности.
